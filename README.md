# Проект «Кулинарные рецепты и списки покупок»

Это проект по курсу «Технологии программирования» (НИУ ВШЭ, БИ). Он демонстрирует работу с ООП на Python, автоматическое тестирование с помощью pytest и культуру работы с Git.

## Что умеет программа

В коде реализованы четыре связанных класса для управления рецептами:
1. **Ingredient (Ингредиент)** — хранит название, количество и единицу измерения. Не позволяет вводить отрицательный вес и умеет сравнивать продукты.
2. **Recipe (Рецепт)** — собирает ингредиенты, автоматически суммирует дубликаты, считает количество уникальных продуктов и умеет масштабировать порции, создавая копию рецепта.
3. **ShoppingList (Список покупок)** — объединяет продукты из разных рецептов, складывает одинаковые ингредиенты и выдает финальный список, отсортированный по алфавиту.
4. **DietaryRecipe (Диетический рецепт)** — наследник обычного рецепта, который добавляет категорию диеты (например, Веганский, Кето) к названию.
